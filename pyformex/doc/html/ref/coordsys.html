22. coordsys — Coordinate Systems.¶

22.1. Classes defined in module coordsys¶

class coordsys.CoordSys(rot=None, trl=None, points=None)¶

A Cartesian coordinate system in 3D space.

The coordinate system is stored as a rotation matrix and a translation vector.

It can be initialized from a rotation matrix and translation vector, or from a set of 4 points: 1 point on each of the axes, plus the origin.

Parameters:

  • rot: rotation matrix (3,3): default is identity matrix.

  • trl: translation vector (3,) or (1,3): default is [0,0,0].

  • points: Coords-like (4,3): if specified, rot and trl arguments are ignored and are determined from:

    rot = :func:`arraytools.normalize`(points[:3] - points[3])
trl = points[3]

Example:

>>> C = CoordSys(points=[
...    [ 4., 0., 0. ],
...    [ 0., 2., 0. ],
...    [ 0., 0., 3. ],
...    [ 0., 0., 0. ]])
>>> print(C.points())
[[ 1.  0.  0.]
 [ 0.  1.  0.]
 [ 0.  0.  1.]
 [ 0.  0.  0.]]
>>> print(C.origin())
[ 0.  0.  0.]
